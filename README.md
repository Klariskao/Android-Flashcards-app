# 📚 Android Flashcards App

An Android application designed to help users study efficiently using digital flashcards.  
The app allows users to create, review, and manage flashcards for learning and memorization.

This project demonstrates Android development fundamentals including UI design, data handling, and interactive user interfaces.

---

# 🚀 Overview

Flashcards are a proven learning technique that relies on **active recall**, helping users improve memory retention by repeatedly testing knowledge.

This Android app provides a simple and intuitive platform for studying with flashcards directly on a mobile device.

The goal of this project was to practice:

• Android UI development  
• Activity navigation  
• Data structures for flashcard management  
• Interactive learning interfaces  

---

# ✨ Features

- 📇 Create custom flashcards
- 🔄 Flip cards to reveal answers
- 🗂 Organize study cards
- 📱 Simple and clean mobile interface
- ⚡ Lightweight and fast Android application
